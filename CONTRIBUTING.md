# 贡献指南

感谢您对 ESP32-MCP 项目的关注！我们欢迎任何形式的贡献。

## 如何贡献

### 报告问题

如果您发现了 bug 或有功能建议，请在 GitHub Issues 中提交。

### 提交代码

1. Fork 本仓库
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

### 代码规范

- 遵循 PEP 8 编码规范
- 添加适当的注释和文档字符串
- 为新功能编写测试
- 确保所有测试通过

### 插件开发

如果您想开发新插件，请参考 [插件开发指南](docs/plugins/README.md)。

## 行为准则

请尊重所有贡献者，保持友好和专业的态度。
